# An Analysis on the Effectiveness of Machine Learning in Predictive Maintenance of Electric Vehicles in the Automobile Industry

## Overview
This project analyses the effectiveness of machine learning techniques for predictive maintenance of electric vehicles using secondary quantitative data. The objective is to identify the best-performing model for predicting maintenance needs and improving vehicle reliability.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib

## Machine Learning Models
- Logistic Regression
- Random Forest
- Support Vector Classifier (SVC)
- XGBoost
- Extra Trees Classifier

## Dataset
The project uses a secondary quantitative dataset containing engine and vehicle-related information.

## Results
The models were trained and evaluated using standard machine learning techniques to compare their predictive performance for electric vehicle maintenance.

- `Artefact.ipynb` – Jupyter Notebook
